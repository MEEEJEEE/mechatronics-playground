## 1. ADC 변환 3단계 ★출제

```
① 샘플링 (Sampling) → ② 양자화 (Quantization) → ③ 인코딩 (Encoding)
```

### ① 샘플링

- 연속 신호를 일정 간격 T로 이산 샘플로 변환
- **나이퀴스트-섀넌 정리:** `f_sample ≥ 2 × f_max`
- 언더샘플링 시 → **Aliasing(앨리어싱)** 발생

### ② 양자화

- 샘플값을 n비트 정수로 변환
- 분해능 = `V_ref / 2ⁿ`
- 양자화 오차: `±(1/2) × 분해능`
- 비트 수 ↑ → 정밀도 ↑ → 양자화 오차 ↓

## 2. ADC 방식 비교 ★중요

| 방식 | 속도 | 정밀도 | 원리 | 용도 |
|---|---|---|---|---|
| **Flash ADC** | 가장 빠름 | 낮음 | 2ⁿ개 비교기 병렬 사용 | 고속 영상처리 |
| **SAR (Successive Approximation)** | 중간 | 중간 | 이진탐색 방식, 상위 비트부터 결정 | 가장 범용 (MCP3002) |
| **Sigma-Delta (ΣΔ)** | 느림 | 가장 높음 | 과샘플링 + 노이즈 쉐이핑 | 오디오, 계측 |

**SAR ADC 동작 원리:** V_ref/2와 비교 → MSB 결정 → 다음 비트로 이진탐색 반복

```
예) 6.8V, 범위 10V, 6bit:
  5V(set 1) → 7.5V(set 0) → 6.25V(set 1) → ...
```

## 3. DAC (Digital → Analog) ★계산

```
Vout = V_ref × D / 2ⁿ
  (D = 디지털 입력값, n = 비트 수)

예) 10bit ADC, V_ref=5V, D=512
  → Vout = 5 × 512/1024 = 2.5V
```

## 4. 2진수 변환 ★출제·계산

### 2진수 → 10진수
```
각 자리 × 2ⁿ 합산
예) 1011₂ = 1×8 + 0×4 + 1×2 + 1×1 = 11₁₀
```

### 10진수 → 2진수
```
2로 반복 나누기, 나머지를 역순으로
예) 13: 13÷2=6 나머지1, 6÷2=3 나머지0, 3÷2=1 나머지1, 1÷2=0 나머지1
  → 역순: 1101₂
```

### 2의 보수 (Two's Complement) — 음수 표현
```
방법: 비트 전체 반전(NOT) 후 +1
예) +5 = 0101 → 반전: 1010 → +1: 1011 = −5

검증: 0101 + 1011 = 0000 (올림 무시) = 0  ✓

장점:
  - 덧셈 회로를 그대로 사용 가능
  - 0이 유일 (Sign-Magnitude는 +0, -0 두 가지 존재)
```

### 이진수 연산 규칙
```
덧셈: 1+1 = 10₂ (0 쓰고 1 올림)  예) 1011 + 0110 = 10001
뺄셈: 빌려올 때 상위 비트에서 2를 현재 자리로   예) 1010 - 0011 = 0111
```

## 5. MCP3002 — 실습 연계

- 10비트 SAR ADC. SPI 통신. Raspberry Pi와 연결.

| 핀 | 이름 | 기능 |
|---|---|---|
| 1 | CS/SHDN | 칩 선택. Low일 때 활성화 |
| 2 | CH0 | 아날로그 입력 채널 0 |
| 3 | CH1 | 아날로그 입력 채널 1 |
| 4 | Vss | GND |
| 5 | DIN | SPI 데이터 입력 (RPi → MCP) |
| 6 | DOUT | SPI 데이터 출력 (MCP → RPi) |
| 7 | CLK | SPI 클록 |
| 8 | Vdd/Vref | 전원 + 기준전압 (2.7~5.5V) |

**RPi 연결:** 핀23 → CLK, 핀21 → DOUT, 핀19 → DIN, 핀24 → CS/SHDN

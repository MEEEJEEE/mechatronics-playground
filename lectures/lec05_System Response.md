| 디지털 | HIGH / LOW 두 상태만 |

## 2. Fourier Series (푸리에 급수) ★출제

주기 신호를 사인·코사인의 합으로 표현. 모든 주기 신호는 사인파의 합으로 분해 가능.

```
F(t) = C₀ + Σ Aₙcos(nω₀t) + Σ Bₙsin(nω₀t)

ω₀ = 2π/T = 2πf₀  (기본 각주파수)
C₀ = DC 성분 (직류 오프셋)

사각파 예시:
F(t) = (4/π)sin(ω₀t) + (4/3π)sin(3ω₀t) + (4/5π)sin(5ω₀t) + ...
→ DC와 홀수 배수 주파수(ω₀, 3ω₀, 5ω₀...)에만 성분 존재
```

---

## 3. Fourier Transform (푸리에 변환)

비주기 신호로 확장. 시간 도메인 ↔ 주파수 도메인 변환.

```
X(ω) = ∫ x(t)·e^(−jωt) dt

Parseval 정리: ∫|x(t)|²dt = (1/2π)∫|X(ω)|²dω  (에너지 보존)
```

## 4. Laplace Transform (라플라스 변환) ★출제·계산

Fourier Transform의 일반화. `s = σ + jω` (복소 변수). 미분방정식 → 대수방정식 변환.

```
F(s) = ∫₀^∞ f(t)·e^(−st) dt
```

### Laplace 변환 쌍 (암기 필수)

| f(t) (시간 도메인) | F(s) | 비고 |
|---|---|---|
| δ(t) — 임펄스 | 1 | |
| u(t) — 단위 계단 | 1/s | |
| t | 1/s² | 램프 함수 |
| **e^(−at)** | **1/(s+a)** | ★ 지수 감쇠 |
| sin(ωt) | ω/(s²+ω²) | |
| cos(ωt) | s/(s²+ω²) | |
| **f'(t) — 미분** | **sF(s) − f(0)** | ★ 초기조건 자동 반영 |
| ∫f(t)dt — 적분 | F(s)/s | |

### 활용 예시

```
예1) y'(t) + 2y(t) = 1,  y(0) = 0
  → s·Y(s) + 2Y(s) = 1/s
  → Y(s) = 1/[s(s+2)] = (1/2)·(1/s) − (1/2)·(1/(s+2))
  → y(t) = (1/2)(1 − e^(−2t))

예2) f(t) = e^(−3t)·u(t)
  → F(s) = 1/(s+3)
```

## 5. 주파수 응답과 필터 ★출제·계산

전달함수 `H(jω) = Vout(jω) / Vin(jω)` — 시스템의 주파수별 응답

### LPF (Low Pass Filter) — RC 회로 ★기출 계산

```
|H| = 1 / √(1 + (ω·RC)²) = 1 / √(1 + (ω/ωc)²)

ωc = 1/(RC)  →  fc = 1/(2πRC)  [차단 주파수]

ω < ωc: 통과 (|H| ≈ 1)
ω > ωc: 차단 (|H| 감소)

−3dB 주파수 = ωc 에서 |H| = 1/√2 ≈ 0.707
```

**기출 계산:** `fc=3.2kHz, C=50μF, π=3.14, 1/π=0.32`
```
R = 1/(2πfcC) = 0.32/(3200 × 50×10⁻⁶) = 0.32/0.16 = 2Ω
```

### HPF (High Pass Filter)

- ωc 이상 통과, 이하 차단. LPF와 반대 특성.

## 6. IIR 필터 (디지털 LPF) ★출제

RC LPF를 디지털로 이산화한 것. Infinite Impulse Response. EMA(지수 이동 평균)와 동일 구조.

```
Vo[n] = (1−α)·Vo[n−1] + α·Vin[n]

α = T/(RC+T)       (T = 샘플링 주기)
1−α = RC/(RC+T)
```

| α 값 | 의미 | 필터 응답 | fc |
|---|---|---|---|
| α 크다 (→1) | 현재 입력에 많이 반응 | 빠른 응답, 노이즈 많음 | 높음 |
| α 작다 (→0) | 이전 출력을 많이 유지 | 느린 응답, 노이즈 적음 (평활화) | 낮음 |

**기출 연계:**
- α 증가 → fc 증가 (α = T/(RC+T), fc=1/(2πRC) → RC 감소 → α 증가)
- 필터 출력이 빠르게 반응하려면 → α 증가 → fc 증가

## 7. 시스템 응답 — 0차/1차/2차 ★출제

### 0차 시스템 (Zero-Order)
```
Vout = K·Vin     →    H(s) = K  (즉각 반응, 지연 없음)
```

### 1차 시스템 (First-Order)
```
H(s) = K/(τs + 1)

τ (시정수, Time Constant): 계단 입력 후 최종값의 63.2%에 도달하는 시간
5τ에서 최종값의 약 99% 도달 (정착 완료)
예) RC 회로: τ = RC
```

### 2차 시스템 (Second-Order)
```
H(s) = K·ωn² / (s² + 2ζωn·s + ωn²)

ωn = 자연 주파수 (Natural frequency)
ζ (zeta) = 감쇠비 (Damping ratio)
```

| ζ 범위 | 이름 | 응답 특성 |
|---|---|---|
| ζ > 1 | 과감쇠 (Overdamped) | 진동 없음, 느리게 수렴 |
| ζ = 1 | 임계감쇠 (Critically Damped) | 진동 없이 가장 빠르게 수렴 |
| 0 < ζ < 1 | 부족감쇠 (Underdamped) | 진동(overshoot)하며 수렴 |
| ζ = 0 | 무감쇠 (Undamped) | 영원히 진동 |

## 8. PID 제어기 ★출제

```
u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt

e(t) = 오차 = 목표값 − 현재값
```

| 파라미터 | 역할 | 증가 시 효과 |
|---|---|---|
| **Kp** | 비례 (현재 오차) | 응답 빠름, overshoot 증가 |
| **Ki** | 적분 (과거 오차 누적) | 정상상태 오차 제거, 진동 가능 |
| **Kd** | 미분 (미래 오차 예측) | overshoot 감소, 노이즈에 민감 |

**기출 답:**
- 응답시간 줄이기 → **Kp 증가**, Ki 증가
- 오버슈트 줄이기 → **Kp 감소, Kd 증가**, Ki 감소

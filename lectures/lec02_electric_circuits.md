## 1. 전압과 전류 기본

| 개념 | 단위 | 정의 |
|---|---|---|
| 전류 (Current) | A = C/s | `i = dq/dt` (단위 시간당 흐르는 전하량) |
| 전압 (Voltage) | V = J/C | 두 점 간 전하 이동에 필요한 에너지. 기준점(GND) = 0V |

---

## 2. 기본 소자 특성 ★계산

| 소자 | V-I 관계 | 임피던스 Z | 위상 | DC 등가 |
|---|---|---|---|---|
| 저항 R | V = IR | R | 0° (동위상) | R 그대로 |
| 커패시터 C | i = C·dV/dt | 1/jωC | 전류가 전압보다 90° 앞섬 | 개방회로 |
| 인덕터 L | V = L·di/dt | jωL | 전압이 전류보다 90° 앞섬 | 단락회로 |

- 커패시터: 전하 저장 (에너지 `Wc = ½CV²`)
- 인덕터: 자기장에 에너지 저장 (에너지 `WL = ½Li²`)
- 저항: 에너지 소산 (열 변환)

## 3. 저항 컬러 코드

- 4밴드 저항: `R = ab × 10^c ± tolerance`
- 색상 값: 검(0) 갈(1) 빨(2) 주(3) 노(4) 초(5) 파(6) 보(7) 회(8) 흰(9)
- 허용 오차: 금(±5%), 은(±10%)
- **포텐셔미터(Pot):** 가변 저항. trimmer, trim pot이라고도 함

## 4. 직렬·병렬 연결

```
저항
  직렬: R_total = R1 + R2 + ...
  병렬: 1/R_total = 1/R1 + 1/R2  →  2개: R1·R2/(R1+R2)

커패시터
  직렬: 1/C_eq = 1/C1 + 1/C2
  병렬: C_eq = C1 + C2  ← 저항과 반대!

인덕터
  직렬: L_eq = L1 + L2
  병렬: 1/L_eq = 1/L1 + 1/L2
```

## 5. 키르히호프 법칙 (KVL / KCL) ★출제

| 법칙 | 공식 | 의미 |
|---|---|---|
| KVL (전압) | ΣVi = 0 | 닫힌 루프에서 전압의 합 = 0 (에너지 보존) |
| KCL (전류) | ΣIi = 0 | 노드에서 들어오는 전류 = 나가는 전류 (전하 보존) |

- KVL 부호 규칙: 루프 방향으로 돌 때 전압이 올라가면 +, 내려가면 −

## 6. 전압 분배기 (Voltage Divider) ★출제·계산

```
Vout = Vin × R2 / (R1 + R2)

예) Vin=20V, R1=3kΩ, Vout=5V → R2 구하기
    5 = 20 × R2/(3k+R2)  →  15k = 15·R2  →  R2 = 1kΩ
```

> **기출 T/F:** 전압 측정은 반드시 **병렬(Parallel)** 연결. 직렬 연결하면 오답.

## 7. Thevenin / Norton 등가회로 ★출제·계산

| 구분 | 구성 | 연결 방식 |
|---|---|---|
| Thevenin | V_th (개방전압) + R_th (등가저항) | 직렬 |
| Norton | I_N (단락전류) + R_N (등가저항) | 병렬 |

**Thevenin 구하는 순서:**
1. 부하 저항 R_L 제거
2. 개방단 전압 측정 → V_th
3. 독립 전원 제거 후 저항 측정 → R_th (전압원→단락, 전류원→개방)

**변환 관계:**
```
I_N = V_th / R_th
R_N = R_th
```

## 8. AC 해석 — 임피던스와 페이저

```
임피던스
  Z_R = R
  Z_L = jωL
  Z_C = 1/jωC = -j/ωC
  일반: Z(jω) = R + jX  (R=저항성분, X=리액턴스)
  직렬 RC: |Z| = √(R² + (1/ωC)²)

오일러 공식 (페이저 기초)
  e^(jθ) = cosθ + j·sinθ
  A·cos(ωt+θ) → 페이저 A∠θ
  AC 회로에서도 옴의 법칙 적용: V = Z · I
```

**전력:**
```
순시 전력: p(t) = v(t)·i(t)  [W]
평균(유효) 전력: P = V_rms · I_rms · cosφ  [W]
V_rms = V_peak / √2 ≈ V_peak / 1.414
cosφ = 역률 (Power Factor), φ = 전압-전류 위상차
```

## 10. 납땜 (Soldering) ★출제

**순서:** 부품·패드 가열(2~3초) → 납 추가 → 계속 가열(1~2초) → 식힘 (불지 말 것)

| 상태 | 설명 |
|---|---|
| **Perfect** | 광택 있는 원뿔형, 핀 주변을 완전히 감쌈 |
| Cold Joint | 불충분한 열. 흐릿하고 거친 표면. 전기적으로 불안정 |
| Too Much Solder | 납이 너무 많아 공처럼 둥글어짐 |
| Insufficient Wetting | 납이 패드/핀에 제대로 붙지 않음 |
| Short | 인접 핀/패드끼리 납 연결 → 단락 발생 |

> IC 소켓 사용 이유: 개발 단계에서 IC를 쉽게 교체하기 위해 (기출 T/F: TRUE)

## 11. 기타 소자

| 소자 | 핵심 내용 |
|---|---|
| 다이오드 | 한 방향 전류. Forward voltage drop ≈ 0.7V. Bridge Rectifier: 4개 → AC→DC 정류 |
| LED | 전류 제한 저항 필수. `R = (V_supply - V_LED) / I_max` |
| Crystal | 피에조 전기 소재(석영). 클록 생성에 사용 |
| Transformer | `V_s = (N_s/N_p) × V_p`. 전압 변환 + 절연 |
| 커패시터 디커플링 | AC 리플 평탄화 → 전원 노이즈 제거 |

## 12. Level Shifting ★출제

5V → 3.3V 변환 3가지 방법:

1. **전압 분배기 (Voltage Divider):** R1, R2로 분압. 단, 출력 임피던스 높아짐
2. **레벨 시프터 IC:** 전용 IC (예: TXB0108). 양방향 변환 가능
3. **Zener Diode / 다이오드 클램핑:** 특정 전압 이상 차단
